<h1 align="center">⚙️ Plataforma Inteligente de Mantenimiento Predictivo</h1>


<h2>📌 Descripción</h2>

<p>
La <b>Plataforma Inteligente de Mantenimiento Predictivo</b> es un sistema diseñado para 
simular el monitoreo de sensores industriales y analizar su comportamiento para detectar 
posibles fallas en maquinaria.

El sistema permite gestionar sensores, analizar datos en tiempo real y construir configuraciones 
de máquinas industriales mediante una arquitectura basada en <b>patrones de diseño</b>, lo que 
garantiza una solución modular, escalable y fácil de mantener.
</p>

<hr>

<h2>🧠 Arquitectura del Sistema</h2>

<p>
La arquitectura del proyecto sigue principios de desarrollo modular y utiliza diferentes 
patrones de diseño para mejorar la organización del código y facilitar la expansión futura 
del sistema.
</p>

<ul>
<li><b>Singleton</b> – Control de una única instancia del motor de análisis predictivo.</li>
<li><b>Factory Method</b> – Creación dinámica de sensores según el tipo solicitado.</li>
<li><b>Abstract Factory</b> – Generación de familias completas de sensores según el entorno.</li>
<li><b>Builder</b> – Construcción paso a paso de configuraciones de máquinas industriales.</li>
</ul>

<hr>

<h2>🛠 Tecnologías Utilizadas</h2>

<ul>
<li><b>NestJS</b> – Framework backend basado en Node.js</li>
<li><b>TypeScript</b> – Lenguaje principal del proyecto</li>
<li><b>Node.js</b> – Entorno de ejecución del servidor</li>
<li><b>REST API</b> – Comunicación entre cliente y servidor</li>
</ul>

<hr>

<h2>⚙️ Funcionalidades Implementadas</h2>

<ul>
<li>Lectura de sensores industriales</li>
<li>Análisis predictivo de valores de sensores</li>
<li>Creación dinámica de sensores</li>
<li>Construcción de configuraciones de máquinas industriales</li>
<li>Arquitectura basada en patrones de diseño</li>
</ul>
